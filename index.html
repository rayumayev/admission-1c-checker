    Исправления (ПН и ВИ)

    Здесь те же столбцы, что в загруженных Excel: автоматические правки (пробелы, название КГ, льготы, квалификация и др.). Ячейки с изменениями подсвечены; наведите курсор, чтобы увидеть исходное значение.

    Сначала выполните проверку. На вкладке «Загрузка» выберите все файлы и нажмите «Проверить» — затем здесь появятся таблицы ПН и ВИ с подсветкой изменённых ячеек.

    Проверяемые критерии

    Отчет формируется по каждому критерию в виде спойлера. Внутри можно отмечать исправленные пункты чекбоксом.

    Активные критерии

    Файл План набора

    • Соответствие столбцам — сверка заголовков файла ПН с шаблоном ПН.
    • Корректность названий КГ — проверка токенов КГ по форме обучения, финансированию и квотам.
    • Код направления в названии КГ — код из столбца `Код` должен присутствовать в названии КГ.
    • Уровень образования в названии КГ — проверка сокращения уровня (например, `БВО`, `БАК`) в КГ.
    • Правила иностранных КГ — для КГ с `ИН` проверяются поля гражданства (Да/Нет).
    • Проверка столбца Только для граждан РФ — значение должно быть `Нет`.
    • Проверка названий льгот — контроль корректных формулировок льгот по правилам.
    • Целевая детализированная квота — для КГ с `|Ц|` должно быть `Да`, иначе `Нет`.
    • Квалификация по уровню образования — для бакалавриата `Бакалавр`, для магистратуры `Магистр`, для остальных `-`, пустое значение недопустимо.
    • Сверка ПН и ВИ — сравнение ПН и ВИ по ключевым полям и фиксация отсутствующих КГ в ВИ.

    Файл Вступительные испытания

    • Соответствие столбцам — сверка заголовков файла ВИ с шаблоном ВИ.
    • Минимальные баллы ВИ — эталон из `min_ball.txt` для обычных предметов и из `spo_min_ball.txt` для предметов СПО; учитывается столбец `Заменяемый предмет`.
    • Максимальные баллы ВИ — проверка по типу `Форма испытания` (ЕГЭ/экзамен/ИД/ИД целевые).
    • Правило 21 балла — контроль типа испытания и пустого `Заменяемого предмета`.
    • Правило 41 балла (СПО) — значение допустимо только для предметов из `spo_min_ball.txt`.
    • Предметы СПО — должны быть указаны только в столбце `Заменяемый предмет`, не в `Предмет`.
    • Русский язык для иностранных граждан — минимальный балл по эталону `min_ball.txt` (как у прочих предметов из этого файла).
    • Формулировки особой отметки — проверка по правилам замен формулировок.
    • Квалификация по уровню образования — для бакалавриата `Бакалавр`, для магистратуры `Магистр`, для остальных `-`, пустое значение недопустимо.
